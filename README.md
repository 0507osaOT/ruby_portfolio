【概要】

飲食店や美容院などの予約を管理できる Web アプリケーションを実装しました。ユーザーは自分の予約を作成・確認でき、管理者は全ての予約を管理できます。
予約時には自動でメールが送信され、前日にはリマインダーメールも届きます。

◾️ 主な機能

・予約管理: 　　　予約の作成・編集・削除、一覧表示

◾️ 使用技術・ツール

Rails 8.0.3 / PostgreSQL / Devise / Pundit / Ransack / Kaminari / Resend API / Solid Queue

◾️Resend API の設定

このアプリケーションはメール送信に[Resend API](https://resend.com/)を使用しています。

● メール通知機能について

このアプリケーションでは、予約に関する重要な情報をメールで自動的にお知らせします。

・届くメールの種類

〈メールの種類〉　　　　　　〈届くタイミング〉　　　　　　　〈内容〉

予約確認メール　　　　　　　　予約を作成した直後　　　　　　　予約内容の確認（日時、名前、連絡先など）

予約変更通知メール　　　　　　予約内容を変更した直後　　　　　変更後の予約内容

予約キャンセル通知メール　　　予約をキャンセルした直後　　　　キャンセルの確認

予約リマインダーメール　　　　予約日の前日　　　　　　　　　　予約の再確認（忘れ防止）


● メールが届かない場合

※以下をご確認ください：

・迷惑メールフォルダに振り分けられていないか確認してください

・登録したメールアドレスに間違いがないか確認してください

・メールの受信設定で、本アプリケーションからのメールが受信拒否されていないか確認してください
